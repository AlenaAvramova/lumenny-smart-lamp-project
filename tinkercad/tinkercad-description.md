# Tinkercad Circuit – Lumenny Smart Lamp

This circuit was created in Tinkercad using Arduino.

## Components:
- Arduino Uno
- PIR Motion Sensor
- LDR (photoresistor)
- LED (lamp simulation)
- Resistors
- Breadboard
- Jumper wires

## How it works:
- The PIR sensor detects motion.
- The LDR measures ambient light.
- The lamp (LED) turns on only when:
  - motion is detected
  - AND it is dark

This ensures energy efficiency and smart behavior.
